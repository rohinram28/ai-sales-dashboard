# AI Sales Dashboard 📈

A lightweight, powerful Streamlit dashboard that analyzes sales datasets to generate instant KPIs, interactive charts, and AI-powered executive summaries.

**🌐 Live Demo:** [Click here to view the live application](https://rohin-ai-dashboard.streamlit.app/)
Note: First download the data "ai_business_sales_dataset.csv" avialable within the project folder.

## ✨ Features
* **Automated KPIs:** Instantly calculates revenue, profit, and margins.
* **Interactive Charts:** Slice data by region and category using Altair.
* **AI Briefings:** Generates tailored business reports using Google Gemini 2.5 Flash.
* **Fault-Tolerant:** Includes fallback architecture to maintain uptime during API rate limits or server outages.
